# Hospital Management Dashboard – Power BI

**Project Overview**

This project is an interactive Power BI dashboard built using the Hospital Management Dataset from [Kaggle](https://www.kaggle.com/datasets/kanakbaghel/hospital-management-dataset).
The goal is to analyze hospital performance, operational efficiency, treatment costs, and billing status to support data-driven decision making in healthcare environments.

The dashboard is designed for hospital administrators, management teams, and analysts.

## Objectives

- Monitor hospital trends and patient flow
- Evaluate doctor and department performance
- Analyze treatment costs and cost drivers
- Track billing status and financial performance
- Identify patterns that may improve efficiency and patient outcomes

## Dashboard Pages
1. **Hospital Trend Overview**
- Provides a high-level view of hospital activity over time:
- Total patients and treatments
- Admissions and treatments trends
- Average treatment cost

Purpose: Quickly understand overall hospital performance.

2. **Doctors/Departments Performance**
- Analyzes performance by medical staff and departments:
- Treatments per doctor
- Average cost per doctor/department
- Patient distribution across departments
- Comparative performance metrics

Purpose: Identify high-performing doctors/departments.

3. **Treatment Cost Analysis**
- Focuses on financial insights related to treatments:
- Cost distribution by treatment type
- Average and total treatment costs
- Cost trends over time
- High-cost treatments and outliers

Purpose: Support cost control and budgeting decisions.

4. **Billing Status**
- Examines hospital revenue and billing efficiency:
- Paid vs. unpaid bills
- Outstanding balances
- Billing trends over time
- Financial status

Purpose: Improve revenue cycle management and cash flow visibility.

## Dataset
Source: Kaggle – Hospital Management Dataset
Tables used:
- Patients
- Doctors
- Treatments
- Appointments
- Billing

Note: The dataset is **synthetic** and used for learning and demonstration purposes only.

## How to Use
1. Download the .pbix file from this repository
2. Open it using Power BI Desktop
3. Refresh data (optional)
4. Explore the dashboard using slicers and filters
